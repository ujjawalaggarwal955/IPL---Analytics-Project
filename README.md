# IPL Analytics Project 🏏

## Overview
This project analyzes IPL cricket data using Python and Pandas to extract meaningful insights about player performance and match impact.

---

## Objectives
- Analyze player performance across IPL seasons
- Identify top batsmen using custom metrics
- Perform phase-wise analysis (Powerplay, Middle, Death overs)
- Evaluate match-winning impact using Player of Match data

---

## Tech Stack
- Python
- Pandas
- NumPy
- Jupyter Notebook

---

## Key Steps Performed
- Data Cleaning and Preprocessing
- Merging Match and Ball-by-Ball Data
- Feature Engineering (Strike Rate, Phase Classification)
- Performance Score Calculation
- Phase-wise Analysis
- Match Impact Analysis

---

## Key Insights
- Different players dominate different match phases
- Performance score balances consistency and scoring rate
- Match-winning players differ from high run scorers
- Death over specialists differ from powerplay players
